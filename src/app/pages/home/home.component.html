Wurm Tools

About these tools

Wurm is an old-school RPG game with a lot of depth where a lot of the mechanics rely on feeling and experiments people have done trying to explain the logic behind it.

Since I love to reverse engineer stuff as a way to learn how others have built software in the past, I have created a small compendium of tools that exploit this knowledge in a direct and simple way.

These tools have not been verified by external sources, so take everything mentioned here with a pinch of salt.

Beware that by using these tools, you may be spoiling yourself some of the fun that comes with discovering the processes!
